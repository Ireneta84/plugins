# Canva Sharing — Detailed Guide

## Share Link Types

### View-only link
- Anyone with the link can view (no Canva account needed)
- Cannot edit, cannot download unless "download" is enabled
- Use for: showing drafts to clients, internal review

### Edit link
- Anyone with the link can edit the design
- Requires a Canva account (Free is fine)
- Changes affect the shared design immediately — no version control
- Use for: collaborative editing with trusted people

### Template link ("Use this template")
- Recipient gets their own copy of the design
- Original is not modified
- Recipient needs a Canva account (Free is fine)
- Brand kit is NOT transferred — styles are embedded as fixed values
- Use for: delivering templates to Free clients, distributing social post templates

### Brand template (Teams only)
- Lives inside a Teams workspace
- Recipient fills in guided data fields
- Requires both parties to be in the same Teams account
- Use for: internal teams, not external client delivery

## Brand Kit Sharing — The Key Constraint

Brand kits are plan-gated:
- **Free users cannot see, apply, or benefit from brand kits** even when viewing a shared design
- The design renders correctly for them (colors show), but they cannot switch to the brand kit or apply it to new designs
- If a Free user edits a shared design, they may accidentally overwrite brand kit elements with non-brand colors/fonts

**The solution for Free clients**: Deliver template links with styles embedded (not via brand kit). The templates look correct and the client can edit their copy, but the brand kit connection is severed — which is fine for clients who just need to use the templates, not manage the brand.

## Folder Sharing

- Folders can be shared with specific people (by email) or via link
- Shared folder access requires a Canva account
- Free users can access shared folders but cannot use Pro elements inside designs

## Exporting for Non-Canva Clients

When the client doesn't use Canva at all:

### PDF options
- PDF (standard) — good for digital viewing, sharing via email
- PDF (print) — high resolution, includes bleed marks, for professional printing
- PDF preserves all visual styling but is not editable

### Image export
- PNG 2x resolution (2160×2160 for a 1080×1080 design) — good for social posts
- JPG if transparency not needed (smaller files)

### Video export
- MP4 for animated designs, reels, stories with motion

### PowerPoint
- PPTX export makes designs editable in PowerPoint
- Font substitution may occur if the client doesn't have the same fonts installed

## Common Scenarios

### Client has Canva Pro
→ Share folder with "can edit" access
→ They can apply the brand kit themselves
→ Add them as a collaborator if you want real-time co-editing

### Client has Canva Free
→ Share individual designs as "Use this template" links
→ Export brand guide as PDF
→ Create an instruction document showing how to use each template

### Client has no Canva account
→ Export all templates as PNG/PDF
→ Deliver as ZIP or Drive folder
→ Include brand guide PDF with hex codes, font names, usage rules

### Client wants to give templates to their team
→ If they have Teams: make them brand templates in their workspace
→ If they have Pro: share folder with team members
→ If they have Free: share template links to each team member individually
